# 🎒 Tripy

본 프로젝트는 인하대학교 전자상거래 기말 프로젝트입니다.

## 📢 소개

> **서비스명** : Tripy
> **주제** : 새로운
> **팀명** : 1조 띵동나루
> **Front-End** : 김승태, 이동령
> **Back-End** : 박상준
> **주소** : `추후 추가 예정`

## 🌳 Branch Strtegy

Github flow 전략을 사용합니다.

## 🌏 Commit Message Convention

```
type: commit message
```

모든 커밋 메세지는 명령문 (현재시제) 를 사용하여 작성합니다. 모든 커밋 메세지 앞에는 다음의 `type` 을 작성합니다.

- `feat`: 새로운 기능 추가
- `fix`: 버그수정
- `docs`: 문서수정
- `style`: 코드 포매팅 등
- `refactor`: 코드 리팩토링
- `chore`: 기타 작업
